# FPS POC (Godot 4)

Minimal first-person prototype:
- Flat ground plane
- CharacterBody3D player
- WASD movement
- Mouse look (pitch + yaw)

## Controls
- W A S D: Move
- Mouse: Look around
- Esc: Release mouse cursor
- Left click: Re-capture mouse cursor

## Run
1. Open this folder in Godot 4.
2. Press Play Scene or Play Project.
